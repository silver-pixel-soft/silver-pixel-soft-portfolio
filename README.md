## 🌟 Overview

Silver Pixel Soft is a highly interactive, cinematic, and production-ready agency portfolio. It is designed to impress with buttery-smooth **GSAP animations**, inertia-based smooth scrolling via **Lenis**, and a premium dark-mode aesthetic using **Tailwind CSS**.

The project is structured for high scalability, maintainability, and exceptional user experience across all devices.

## ✨ Features

- **Cinematic Preloader:** A custom, GSAP-powered loading sequence that sets the tone for the entire experience.
- **Inertia Smooth Scrolling:** Integrated Lenis scrolling for a native, fluid feel across all browsers.
- **High-End GSAP Animations:** Complex scroll-triggered animations, staggered text reveals, and parallax effects.
- **Custom Cursor:** A bespoke, physics-based custom cursor that interacts with links and buttons.
- **Responsive Design:** Fully fluid layouts optimized for desktop, tablet, and mobile viewing.
- **Modern Routing:** Seamless page transitions using React Router v6.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Ensure you have **Node.js** (v16.0.0 or higher) and **npm** installed on your system.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/silver-pixel-soft-portfolio.git
   ```

2. **Navigate to the frontend directory**
   ```bash
   cd silver-pixel-soft-portfolio/Frontend
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173` to view the application.
